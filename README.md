**Simple React Calculator**

**Introduction**
This is a simple calculator built with React. It performs basic arithmetic operations like addition, subtraction, multiplication, and division.


**Features**
* Addition
* Subtraction
* Multiplication
* Division
* Clear function to reset calculations


**Technologies Used**
* React
* JavaScript
* HTML/CSS
